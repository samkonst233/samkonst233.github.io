# Dashboard

Access the FiveMSecure dashboard at [fivemsecure.com/dashboard](https://fivemsecure.com/dashboard).

## Authentication

Sign in with Discord to access your server configurations.

## Navigation

### Dashboard

View server overview and quick stats.

### Analytics

View ban statistics with charts and time filters.

### Players

View active players with real-time data:
- Player list with health/armor
- Server identifiers
- Position coordinates
- Kick/ban actions

### Monitoring

Watch player screens live with WebRTC streaming.

### Bans

Manage player bans:
- View all bans
- Search by ID or reason
- Unban players

### Admins

Manage dashboard access with custom permissions.

### Console

Access server console to execute commands and view logs.

### Resources

Manage server resources:
- View all resources
- Start/stop/restart
- Filter by status

### Settings

Configure anti-cheat protections and blacklists.

### Master Actions

Import/export configurations.

### Logs

Configure Discord webhook URLs.

## Quick Actions

**Ban a Player**
1. Go to Players or Monitoring
2. Click on player
3. Click Ban
4. Enter reason and duration
5. Confirm

**Kick a Player**
1. Go to Players or Monitoring
2. Click on player
3. Click Kick
4. Enter reason
5. Confirm

**Unban a Player**
1. Go to Bans
2. Find the ban
3. Click Unban
4. Confirm

## Saving Changes

Always click **Save Config** after making changes.
