# Server Console

Execute commands and view logs directly from the dashboard.

## Access

Go to **Dashboard** â†’ **Console**

## Features

- View real-time server logs
- Execute server commands
- Filter logs by type
- Auto-scrolling updates

## Executing Commands

1. Type command in input box
2. Press Enter or click Execute
3. View command output in logs

## Common Commands
```bash
# Restart a resource
restart resource_name

# Stop a resource
stop resource_name

# Start a resource
start resource_name

# List all resources
resources

# Server info
status
```

## Log Types

- **Info** - General information
- **Error** - Error messages
- **Warning** - Warning messages
- **Success** - Success messages

## Tips

- Use up/down arrows for command history
- Commands execute immediately
- Be careful with sensitive commands
