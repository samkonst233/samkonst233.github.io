# FAQ

Frequently asked questions about FiveMSecure.

## General

**What is FiveMSecure?**

FiveMSecure is a complete anti-cheat and monitoring platform for FiveM servers that provides protection against cheats, live player monitoring, and server management tools.

**How much does it cost?**

Visit [fivemsecure.com](https://fivemsecure.com) for current pricing.

**Do I need a separate license for each server?**

Yes, each FiveM server requires its own license.

## Installation

**Does it work with ESX/QBCore?**

Yes, FiveMSecure is framework-agnostic and works with all FiveM frameworks.

**Can I test it before buying?**

Contact support@fivemsecure.com for trial information.

**How do I update FiveMSecure?**

Download the latest version and replace the old files, keeping your config.

## Features

**Can I monitor multiple players at once?**

Yes, you can watch up to 4 player streams simultaneously.

**Does it record streams?**

Screenshot capture is available. Video recording coming soon.

**Can I whitelist specific players?**

Yes, through the dashboard or using Lua exports.

**How accurate is the anti-cheat?**

Start with "Warn" actions and monitor logs to tune for your server.

## Permissions

**Can admins see all servers?**

No, admins only see servers they're granted access to.

**Can I limit what admins can do?**

Yes, you can configure granular permissions for each admin.

## Technical

**Does it affect server performance?**

Minimal impact. Client-side monitoring uses ~5-10 FPS capture.

**What ports does it use?**

- HTTPS: 443 (API)
- UDP: 3001 (Streaming, configurable)

**Is player data encrypted?**

Yes, all connections use SSL/TLS encryption.
