# Installation

This guide covers installing FiveMSecure on your FiveM server.

## Prerequisites

Before starting, ensure you have:

- A running FiveM server (latest version recommended)
- Valid FiveMSecure license key
- Admin access to server files
- Basic knowledge of server.cfg configuration

## Step 1: Download Resource

Download the latest FiveMSecure release from your dashboard at [fivemsecure.com/dashboard](https://fivemsecure.com/dashboard).

## Step 2: Extract Files

Extract the downloaded archive. You should see a folder named `fivemsecure`.
