# Admin System

Grant dashboard access to trusted users with custom permissions.

## Adding Admins

1. Go to **Dashboard** â†’ **Admins**
2. Click **Add Admin**
3. Enter Discord ID and name
4. Configure permissions
5. Click **Add Admin**

## Permissions

Admins can have these permissions:

- View Dashboard
- View Analytics
- View Players
- View Monitoring
- View Bans
- View Console
- View Resources
- Edit Settings
- Use Master Actions
- View Logs
- Manage Admins

## Editing Admins

1. Go to **Admins**
2. Click **Edit** on an admin
3. Change permissions
4. Click **Save Changes**

## Removing Admins

1. Go to **Admins**
2. Click **Remove** on an admin
3. Confirm removal

## Best Practices

- Only grant necessary permissions
- Review admin list regularly
- Remove inactive admins
- Use descriptive admin names
